\# AI-Powered Resume \& Job Matcher



A Streamlit-based application that analyzes a resume against a job description,

calculates percentage match, determines eligibility, identifies missing skills,

and suggests what to learn for the target role.



\## Features

\- Resume PDF parsing

\- Job description analysis

\- Match percentage calculation

\- Eligibility classification

\- Skill gap detection

\- Learning recommendations



\## Run Locally

```bash

python -m pip install -r requirements.txt

streamlit run app.py
